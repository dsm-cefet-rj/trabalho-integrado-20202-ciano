# Biblioteca

### Inicializar o React
`yarn && yarn start`

### Inicializar Json-Server
Entre na pasta "json_server" e execute o comando: `json-server --watch db.json --port 3004`

[Mais Informações sobre Json-Server.](https://github.com/typicode/json-server#json-server--)
